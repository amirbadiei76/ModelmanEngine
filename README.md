# ModelmanEngine
🚀 **ModelmanEngine** helps Iranians search the internet freely, even with restrictions and after the 2026 war – extracting full text, images/GIFs, and all links from **most webpages** (excluding those protected by aggressive bot defenses), and saving a PDF snapshot of each page.

ℹ️ **To get started:** First, either **fork** this repository (for public use) or click the **plus icon** and choose **Import repository** (for private use) to create your own copy. Then, **open that copied repository** in your GitHub account – all workflows below will work from the **Actions** tab.

🛠️ **How to run a workflow:**  
- 💻 **Windows & Android (desktop mode on):** Go to **Actions** → choose your favorite action → **Run workflow ▽** → pick settings → click green **Run workflow**.  
- 📱 **Android (desktop mode off):** Tap **More ▽** → **Actions** → **All workflows ▽** → choose your favorite action → **Run workflow ▽** → pick settings → tap green **Run workflow**.

**Available workflows:**
- ♻️ **Factory reset:** Run **♻️ Factory Reset** → removes all downloaded images and history → back to clean start.  
- 🧹 **Clear History:** Run **🧹 Clear History** → deletes only the `History/` folder (preserves images and README).  
- 🔍 **Search:** Run **🔍 Search** → fetch any URL (example: [https://giphy.com/explore/thor-avengers](https://giphy.com/explore/thor-avengers) — or you can write it without `https://`, like [giphy.com/explore/thor-avengers](https://giphy.com/explore/thor-avengers)). The workflow will prepare images, GIFs, text, links, and a PDF for you. After it finishes, go back to the root of your repository – the `README.md` will show the images and GIFs first, followed by the extracted text and links. The original images and GIFs are saved in the `images/` folder, and the PDF is in the `pdf/` folder.
